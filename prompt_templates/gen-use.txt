EXAMPLE: Calculator DSL
========================

To illustrate what is expected, here's a complete example:

Example Requirements:
R1 - The grammar should support variable declarations with numeric types.
R2 - The grammar should support basic arithmetic operations (add, subtract, multiply, divide).
R3 - The grammar should support function definitions with parameters.
R4 - The grammar should support conditional expressions (if-then-else).
R5 - The grammar should support loops with iteration control.
R6 - The grammar should support comments for documentation.

Example Grammar:
```
Program: statements*=Statement;

Statement:
    VariableDeclaration | Assignment | FunctionDef | IfStatement | Loop | Expression
;

VariableDeclaration:
    'var' name=ID ':' type=Type '=' value=Expression
;

Type: 'int' | 'float';

Assignment:
    name=ID '=' value=Expression
;

FunctionDef:
    'function' name=ID '(' params*=Parameter[','] ')' ':' type=Type
    'begin'
        body*=Statement
    'end'
;

Parameter: name=ID ':' type=Type;

IfStatement:
    'if' condition=Expression 'then'
        then_body*=Statement
    ('else'
        else_body*=Statement)?
    'endif'
;

Loop:
    'for' var=ID 'in' start=Expression 'to' end=Expression
        body*=Statement
    'endfor'
;

Expression:
    Arithmetic | FunctionCall | ID | NUMBER
;

Arithmetic:
    left=Operand op=Operator right=Operand
;

Operand: NUMBER | ID | '(' Expression ')';

Operator: '+' | '-' | '*' | '/';

FunctionCall:
    name=ID '(' args*=Expression[','] ')'
;

Comment: /\/\/.*$/;
```

Example Sample Models (covering requirements):

Model 1 (R1 - Variable declarations):
```
var x : int = 10
var y : float = 3.14
var z : int = 42
```

Model 2 (R2 - Arithmetic operations):
```
var a : int = 5 + 3
var b : int = 10 - 2
var c : int = 4 * 6
var d : float = 20 / 4
```

Model 3 (R3 - Function definition):
```
function add : int
begin
    var result : int = 10 + 5
end

function multiply : int
begin
    var product : int = 3 * 7
end
```

Model 4 (R4 - Conditional expression):
```
var x : int = 10
if x then
    var y : int = 20
else
    var y : int = 30
endif
```

Model 5 (R5 - Loop):
```
for i in 1 to 10
    var temp : int = i * 2
endfor
```

Model 6 (R1 + R2 + R6 - Combined: variables, arithmetic, comments):
```
var a : int = 5
var b : int = 10
var sum : int = a + b
var product : int = a * b
```
