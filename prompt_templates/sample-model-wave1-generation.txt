EXAMPLE: Calculator DSL Wave 1
================================

Example Requirements:
R1 - The grammar should support variable declarations with numeric types.
R2 - The grammar should support basic arithmetic operations (add, subtract, multiply, divide).
R3 - The grammar should support function definitions with parameters.
R4 - The grammar should support conditional expressions (if-then-else).

Example Grammar:
```
Program: statements*=Statement;

Statement:
    VariableDeclaration | Assignment | FunctionDef | IfStatement | Expression
;

VariableDeclaration:
    'var' name=ID ':' type=Type '=' value=Expression
;

Type: 'int' | 'float';

FunctionDef:
    'function' name=ID '(' params*=Parameter[','] ')' ':' type=Type
    'begin'
        body*=Statement
    'end'
;

Parameter: name=ID ':' type=Type;

IfStatement:
    'if' condition=Expression 'then'
        then_body*=Statement
    ('else'
        else_body*=Statement)?
    'endif'
;

Expression: Arithmetic | ID | NUMBER;
Arithmetic: left=Operand op=Operator right=Operand;
Operand: NUMBER | ID;
Operator: '+' | '-' | '*' | '/';
```

Example Wave 1 Models (ONE requirement per model):

Model 1 (R1 ONLY - Variable declarations):
```
var x : int = 10
var y : float = 3.14
```

Model 2 (R2 ONLY - Arithmetic operations):
```
var result : int = 5 + 3
```

Model 3 (R3 ONLY - Function definition):
```
function calculate : int
begin
    var temp : int = 5
end
```

Model 4 (R4 ONLY - Conditional):
```
var x : int = 10
if x then
    var y : int = 20
endif
```

Notice: Each model is SIMPLE and focuses on ONE requirement only.
