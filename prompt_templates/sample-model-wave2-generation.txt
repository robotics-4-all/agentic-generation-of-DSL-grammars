EXAMPLE: Calculator DSL Wave 2
================================

Example Requirements:
R1 - The grammar should support variable declarations with numeric types.
R2 - The grammar should support basic arithmetic operations (add, subtract, multiply, divide).
R3 - The grammar should support function definitions with parameters.
R4 - The grammar should support conditional expressions (if-then-else).
R5 - The grammar should support loops with iteration control.

Example Grammar:
```
Program: statements*=Statement;

Statement:
    VariableDeclaration | Assignment | FunctionDef | IfStatement | Loop | Expression
;

VariableDeclaration:
    'var' name=ID ':' type=Type '=' value=Expression
;

Type: 'int' | 'float';

FunctionDef:
    'function' name=ID '(' params*=Parameter[','] ')' ':' type=Type
    'begin'
        body*=Statement
    'end'
;

IfStatement:
    'if' condition=Expression 'then'
        then_body*=Statement
    ('else'
        else_body*=Statement)?
    'endif'
;

Loop:
    'for' var=ID 'in' start=Expression 'to' end=Expression
        body*=Statement
    'endfor'
;

Expression: Arithmetic | ID | NUMBER;
Arithmetic: left=Operand op=Operator right=Operand;
Operand: NUMBER | ID;
Operator: '+' | '-' | '*' | '/';
```

Wave 1 Models (for reference - these were the simple models):
```
--- Wave 1 Model 1 (R1) ---
var x : int = 10

--- Wave 1 Model 2 (R2) ---
var result : int = 5 + 3

--- Wave 1 Model 3 (R3) ---

end
```

Example Wave 2 Models (COMBINE 2-3 requirements):

Model 1 (R1 + R2 + R4 - Variables, Arithmetic, Conditionals):
```
var x : int = 10
var y : int = 20
var sum : int = x + y
if sum then
    var result : int = sum * 2
else
    var result : int = 0
endif
```

Model 2 (R3 + R2 + R5 - Functions, Arithmetic, Loops):
```
function processData : int
begin
    var total : int = 0
    for i in 1 to 10
        var temp : int = i * 2
    endfor
end
```

Notice: Wave 2 models combine multiple requirements and are more realistic/complex than Wave 1.
