# Validation and Statistical Scripts

This folder contains the focused scripts released with the supplementary artifacts:

- `grammar_validator.py`: validates TextX grammar specifications.
- `model_validator.py`: validates generated DSL models against their target TextX grammar.
- `experiment1_statistical_tests.py`: produces the Experiment 1 statistical analysis.
- `experiment2_statistical_tests.py`: produces the Experiment 2 paired statistical analysis and confidence intervals.
- `requirements.txt`: Python dependencies for these scripts.

The repository intentionally does not include the broader grammar-generation pipeline, LLM client integrations, runtime logs, or orchestration code.
